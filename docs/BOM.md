## 전원

| 부품명 | 모델명 | 역할 | 수량 | 단가 | 합계 |
|--------|--------|------|------|------|------|
| 보드용 배터리 | Li-Po 14.8V (4S) 1000mAh | RPi5, LiDAR 보조전원, Stella, IMU, RFID, 디스플레이 공급 | 1 | | |
| 모터용 배터리 | JIPOWER Li-Po 22.2V (6S) 5200mAh 35C | DC 모터(MD20A), 서보(UBEC 경유) 공급 | 1 | | |
| RPi5 전용 DC-DC 컨버터 (USB-PD 트리거 포함) | 확정 필요 | RPi5 전용 5V/5A 공급 (PD 프로토콜 협상 필요) | 1 | | |
| DC-DC 컨버터 | XL4015 강하형 5A 가변 [SZH-PWSDF-036] | LiDAR USB_PWR(보조전원) 공급 | 1 | | |
| UBEC (서보용) | 확정 필요 | 모터용 배터리(22.2V) → 서보 정격전압(6~7.4V) 강하 | 1 | | |
| 전압 표시 모듈 | FND 전압표시 | 배터리 전압 모니터링 | 2 | | |

> - 보드용/모터용 배터리 전원 라인은 분리하되, 데이터 통신(LiDAR, Arduino 등)을 위해 모든 기기의 GND는 공통으로 연결.
> - 보드용 배터리(1000mAh) 기준 실사용 런타임은 약 35~50분으로 추정되어, README의 "연속 운행 2시간" 목표 대비 용량 증설(3000mAh 이상 권장) 또는 배터리 교체 운용 검토 필요.
> - 42-775B 모터, MD20A 드라이버의 정격 전압이 모터용 배터리(22.2V, 풀충전 시 25.2V)와 맞는지 스펙시트 확인 필요.
